# Lossless-image-compression-algorithms

❗These algorithms are the result of a 2nd-year coursework project.❗

It provides 5 different lossless image compression algorithms in Python:
1. RLE 
2. LZW
3. Huffman
4. Lossless JPEG
5. Deflate

The "входные данные" folder contains the original images. 

the "выходные данные" folder contains the encoded compressed images

The "восстановленные" folder contains the compressed images
